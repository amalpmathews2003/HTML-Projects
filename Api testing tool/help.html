Online REST API Testing Tool
This is an Online API testing tool made by Amal P Mathews .You can use this tool to test your Api directly from your browser.

Now It has GET,POST methods.

Get
  1. Enter the url in URL input section
  2. Set the dropdown menu to GET (If it is on GET no need)
  3. Press the Send button
  4. The response will be shown in the output section in JSON format
Post
  1. Enter the url in URL input section
  2. Set the dropdown menu to POST (If it is on POST no need)
  3. Enter your content in input section in JSON format
  4. Press the Send button
  5. The response will be shown in the output section in JSON format
